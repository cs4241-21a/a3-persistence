How to use the Forms

When submitting a movie, please make sure that all forms are filled out. If not all are filled out you WILL receive an error.

In the event that you have incorrectly added a movie date, please utilize the specific row and either change the attributes of the row or choose to delete. This must be followed up with clicking on the submit button.

As mentioned earlier, if you wish to delete a current MovieHub, click delete in the specified row to remove the movie.
