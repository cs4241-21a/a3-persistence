Task Management Tool

This tool provides the abilities to add/modify/delete a task and helps users learn how many days are remaining to complete the task

Tool Usage

To add a task, fill in all three fields and submit.

To modify a task, click on the modfiy button for the specific row you want to change. Then, type in the existing task name and change the priority or due date and then click submit.

To delete a task, click the delete button on the desired row you want to move.
